-How to run:

Commands:

To Dowload React-Native:
```npm i react-native```

Running Commands
```npm i```
```react-native run-android```

# QRScannerExpo

This is the expo version of the project compared to the vanilla version.

-How to run:

Commands:

To Dowload React-Native:
```npm i react-native```

To Dowload Expo:
```npm install -g expo-cli```
```npm install -g expo```

Running Commands
```npm start```
